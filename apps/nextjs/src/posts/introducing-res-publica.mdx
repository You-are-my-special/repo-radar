Why do you need this? You already have Github issues, Jira, Trello, and a bunch of other tools. Why do you need another one?
This is true. You don't need another one. But you might want one. And that's where Res Publica comes in.


## What is Res Publica?

Res Publica is an open source issue tracker. It's built with Next.js, Tailwind CSS, and MDX. It's designed to be simple, fast, and easy to use. It's also designed to be customizable, so you can make it your own.

## Tech Stack

- Next.js
- Tailwind CSS
- MDX
- Edgedb
